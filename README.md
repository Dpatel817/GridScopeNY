# GridScope_NY

A Streamlit-based NYISO dashboard for market analysis.

## Structure
- `app.py` - main Streamlit entrypoint
- `pages/` - page modules
- `src/` - reusable logic
- `scripts/` - ETL and preprocessing scripts
- `data/raw/` - raw source files
- `data/processed/` - processed outputs

## Quick start

```bash
pip install -r requirements.txt
streamlit run app.py
```
